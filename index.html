Deeeep.io Reef Client

What is DRC?

Deeeep.io Reef Client (DRC) is a new Deeeep.io client that supports v4.

Features

  • Asset Swapper
  • Custom Theme
  • Theme Maker
  • Light Theme
  • Discord Rich Presence
  • Built-in Docassets
  • Better UI
  • Evolution Tree
  • V3 game UI

FAQs

How do I install it?

The easiest way is to go to the releases page and download "Deeeep.io-Reef-Client-Setup.exe", then run it.

Is this safe to open?

Yes, this client is absolutely not a virus and is safe to open. Although Windows or your antivirus may try to block it, its only because you downloaded it from the internet and it doesn't have a verification certificate (those are expensive :c). No known antivirus has ever flagged this as malicious. If you aren't convinced, view the full source code here

Is this open source?

Yes, DRC is fully free and open source, licensed under the GNU GPL, a free (Free as in Freedom) license. You can view the full source code here.
