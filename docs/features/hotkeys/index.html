Hotkeys

About the hotkeys and keyboard shortcuts of the Deeeep.io Reef Client.
Keybinds can be changed by opening the General tab of the Settings, then clicking on the "Edit" button next to "DRC Keybinds".

DevTools (F12, cannot be changed)

Opens DevTools pane. Only for development use.


Cancel Charge (Default KeyC)

Cancels charge boost if charging like right-click. Page: Touchpad


Evolution Tree (Default KeyT)

Opens the Interactive Evolution Tree. This hotkey only works in-game.


Screenshot (Default KeyV)

Takes a screenshot of the game. This hotkey only works in-game.


Ghost (Default KeyX)

If spectating, kill the ghost like Key Q. This hotkey only works if the player is a Ghost.


Alt + C: Copy URL

Copies URL of the current page.


Alt + J: Join Game

Opens a modal that allows you to join a game server either by code or direct URL.
